# 性压抑指数计算器

Sexual Repression Index Calculator

一个基于React的性压抑指数计算器应用。

## 本地运行

```bash
# 使用简单HTTP服务器
node simple-server.js

# 或者使用Python
cd web
python3 -m http.server 3003
```

访问 http://localhost:3003
